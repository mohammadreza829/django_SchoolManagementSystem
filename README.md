## ✅ نحوهٔ اعتبارسنجی

دستورالعمل نصب بهصورت سرتاسری روی **Python 3.12 + Django 6.0.6** اجرا و تأیید شد:

- `python manage.py check` → بدون خطا (۰ ایراد)
- `python manage.py makemigrations --check --dry-run` → بدون تغییر باقیمانده
- `python manage.py migrate` → همهٔ مهاجرتها با موفقیت اعمال شد
- `python seed_demo.py` → دادههای نمونه (۳۶ دوره، ۳۷۷ جلسه، ۱۸ آزمون و ...) ساخته شد
- `runserver` + بررسی صفحات: `/`، `/courses/`، `/quiz/`، `/accounts/login/` → **200** و `/admin/`، `/panel/` → **302** (ریدایرکت به ورود، مطابق انتظار)
